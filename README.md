# SSH Client for Cardputer-adv with ILI9341 LCD panel

This project is based on AndyAi's ILI9341(320x240) LCD support (https://github.com/AndyAiCardputer/zx-spectrum-cardputer-ili9341).

- For connecting the LCD panel, please refer to [AndyAi's YouTube video](https://www.youtube.com/watch?v=ucNRMeOsK64)
- For build code and SSH/Wireguard configuration, please refer to the original [M5-SSH-Wireguard](https://github.com/bpivk/M5-SSH-Wireguard)
- AndyAi's LCD cover is for panels without a touch panel. I redesigned the cover because I was using a thicker panel with touch panel feature. I removed the SPI signal terminals on the LCD panel and wired them directly. Attaching the STL file for reference. The thickness of LCD panel including the SD card slot is designed to be 8.8mm.
